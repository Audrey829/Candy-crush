# Candy Crush Mini Project

## Project Overview
This is a simplified Candy Crush style game built in Java as our mini-term group project.

## Current Goal
Build a basic playable version within one week:
- 9x9 board
- random candy generation
- adjacent swap
- 3-match elimination
- score + moves
- simple win/lose condition

## Project Structure
- `model/`: data classes
- `logic/`: core game rules
- `ui/`: user interface
- `util/`: constants and helper classes
- `docs/`: team planning notes

## Team Plan
- Member A: Board + Candy classes
- Member B: Match finding
- Member C: Move handling / elimination
- Member D: UI and integration

## Running the Project
Run `Main.java` to start the basic test version.
